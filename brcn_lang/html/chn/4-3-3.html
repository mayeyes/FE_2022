淡钓

保宁青川水库就是在忠南地区具有代表性的钓鱼台，非常有名。 它的规模达84万坪，其水库的水质也不错，能钓上条大鱼。 相对成渊池是半峡谷型的， 特别在春季产卵时期能够钓上很多土产鲫鱼。 这两个地方都少有小小鱼， 大部分都是25厘米以上， 主要鱼类是条香鱼

长铉池是山势秀丽， 水也深，还有周边环境安静又舒适， 所以，相信可能满足钓鱼人在 安静的环境里要钓鱼的要求。 并且在雨季时，来自上游的海水就在蓝浦池形成钓鱼台， 在这儿容易钓上6寸以上的大鱼， 因此外行的人也享受到钓鱼。 初此以外，时有钓上有条4尺大鱼的晋竹池池, 由于在水底形成淀粉层， 具有丰富鱼资源的龙楮池等 可说是整个保宁就是钓鱼台。

青萝(青川)水库

  • 水面积 : 840,000 坪
  • 鱼类 : 鲫鱼, 鲤鱼,Opsariichthys bidens, 乌鱼, 香鱼 ,黄颡鱼

概况: 青萝水库位于有西海岸最大海水浴场的保宁市。 它于1960年9月建设的一种U型的半峡谷型水库。它属于秋季的钓鱼台， 因此在10月初钓得最好。由于在堤坝左边(壮山里)的宽度又窄又长，水礁的状态也不太好。 但，其右边(乡川里)的宽度大，水也浅，因此水礁也是均衡地分布水里。那在左边是夏季， 秋季的钓鱼台, 在右边从春季到冬季冷冰时，能够钓鱼。特别在左边(壮山里) 一带就是重要的钓鱼点(point)。

京釜高速公路 -天安 IC->温阳->洪城->大川->36路国道->在青川桥 丁字路口右拐->堤坝右边(乡川里)

  • 大川站 公共汽车站->上青阳 市内公共汽车(水库 右边)
  • 大川站 公共汽车站->上玉桂 市内公共汽车(水库 左边)

真竹 水库

  • 鱼类 : 鲫鱼
  • 钓鱼方法 : 春季在水礁上钓鱼最有效

钓鱼方法 : 春季在水礁上钓鱼最有效。 若使用2.5～3.5段的鱼竿，3～4号圆丝的1.5～1.8厘米的钓鱼线，然后把微小的3-4 条蚯蚓钩在鱼钩上，这样，钓上大鱼的可能性很大。 概况: 从右边上游到堤防的水礁地区和在左边上游松岩洞浅水的水礁是在一年里一 直有名的钓鱼点(point)，在这儿能钓到大量25～30厘米的鲫鱼

长铉池水库

因青萝的名声而长铉池不太闻名，但在数量方面并不亚于青萝池。

它的规模达3万3千坪，是一种峡谷型的水库，长铉池是山势秀丽，水也深， 还有周边环境安安静静，又舒适，所以，相信可能满足钓鱼人在安静的环境里要钓鱼的要求。 主要的鱼类是条7-8寸的土产鲫鱼，在春季产卵期或夏季夜里钓鱼时候，能期待钓到大鱼的愿望。 这儿的鲤鱼因受到干净水质的影响，所以拉力很大，把浮标的往上顶的力量也很大。 鲤鱼也是有60厘米以上的条大鱼，栖息的密度相高。糕点的钓鱼就最普遍的，可是， 蚯蚓诱饵钓鱼也好。 据当地人说， 若蚯蚓等用小动物的钓鱼，反而能钓上大鱼的机率增大。

长铉池是在大川站上青阳方向的36国道以后，到独亭丁字路口往左拐。 然后在玉桂丁字路口往 长铉去. 沿着耕地旁边的里再上2 公里就到长铉池， 在这儿再沿着小路要1公里就到明代峡谷入口。 在大川车站上青萝明代方向公共汽车 青萝 明台峡谷方向公共汽车 09:25~17:40 4回, 约要 40分钟

蓝浦池水库

蓝浦池是作为代表忠南的个钓鱼台，许多钓鱼的纷纷找来的。 以前往往见到钓上大鱼的人，但，现在大多是6寸以上中大小的少数量。 鱼的拉力很大，把浮标的往上的力量亦很大。在雨季时，因从上游流水而形成 个新钓鱼点(point),从而在水面，水色平静以后，就在堤防右边沿岸上游的突出部分钓 鱼也不错 蓝浦池在大川往舒川方向沿着21路国道就到保宁市综合操场， 然后再沿着国道要3公里就到蓝浦面，在其入口往右拐(月颠方向)要1公里，就在后左边。

龙楮池水库

由于在水底形成淀粉层，既没有水干的现象，也具有丰富鱼资源。 那儿有鲫鱼, 鲤鱼,乌鱼 等鱼类，而且在堤防的左边，田地及最上游的芦田就是一 个最佳钓鱼点(point)。

龙楮池是在保宁市内往舒川方向一直走，在经过熊川邑到珠山面时， 跟珠山的村民可以问路。
