구분 운영일
성수기 06.01 ~ 08.31(3개월)
난방기 09.01 ~ 익년 05.31(9개월)
  • 난방 기간 중 주중 대여불가(월, 화, 수, 목, 일)
  • 성수기예약 접수일 : 매년 5월 1일(인터넷 예약 접수)
  • 본인 미동행 시 금액 상향 시킴(신분증 확인 후 미동반 시 객실당 1만원 추가 징수)
  • 인원 초과 시 인당 10,000원, 침구류 추가 대여 시 10,000원 적용
  • 회의실(온돌) 숙박불가
  • 학과 단체행사 시 객실에 한하여 사용료 50% 적용(MT, OT에 한함) 단, 학생복지팀에서 인정한 행사에 한함(집단연수허가원 제출)
  • 식당요금 실당 10,000원 적용
  • 반려동물 출입불가
  • 세면도구 필수 지참
  • 설·추석 명절은 운영 안함
  • 세미나실 및 회의실 사용은 객실 예약자에 한함
